Use this skill whenever the run needs external facts, source discovery, or
evidence gathering.

- Discover sources with `web_search` using focused queries; refine with
  `country`, `language`, or `freshness` when relevant.
- Read the best sources in full with `web_fetch` before citing them — snippets
  are not enough.
- Synthesize as you go, then interrogate the synthesis for gaps: unanswered
  sub-questions, single-source claims, and contradictions.
- Close each gap with a sharper follow-up search instead of stopping early.
- Cite the URLs you actually read in the final answer.
