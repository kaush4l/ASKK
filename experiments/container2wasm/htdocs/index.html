container2wasm spike: Alpine Linux in a browser tab

ASKK feasibility batch 05. This page boots a real OCI container image (alpine:3.18, x86_64) inside a CPU emulator compiled to WebAssembly (container2wasm, the WASI-on-browser path), runs it in a Web Worker, and executes one shell command. The glue JS is vendored from the upstream demo (Apache-2.0); the multi-MB container .wasm chunks are fetched by fetch-assets.sh and are not in git.
